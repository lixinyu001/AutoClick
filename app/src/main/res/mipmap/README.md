# App Icons

This directory should contain the app launcher icons in different resolutions:

- mipmap-mdpi/ (48x48)
- mipmap-hdpi/ (72x72)
- mipmap-xhdpi/ (96x96)
- mipmap-xxhdpi/ (144x144)
- mipmap-xxxhdpi/ (192x192)

For now, the app will use default Android icons. You can replace these with custom icons later.

To add custom icons:
1. Create PNG images in the appropriate sizes
2. Place them in the corresponding mipmap folders
3. Name them: ic_launcher.png and ic_launcher_round.png

Alternatively, you can use Android Studio's Image Asset Studio to generate icons.
